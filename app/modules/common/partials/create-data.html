Create Data

Upload Datasets

Create new datasets, by providing tabular data from any Open Data source.
This allows you to select the suitable data in an interactive way and enrich the dataset information (metadata).

Define Indicators

Datasets are associated with a properties, such as unemplyment, health.
Those properties are used for the creation of metric.

Create metrics

Create new datasets by using the inteactive user interface. Apply datasets in mathematical-like forumlas. Store them as a metric to use them with many different datsets of the same indicator.

Apply Datasets to Metric

Apply real datasets of the respective properties of a metric.
The output will be a new dataset.
