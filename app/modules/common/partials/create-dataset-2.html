Step 2 - Class and Metadata

Your data refers to a specific class type. e.g several couontries. Please select your class type. The instances for a class type, e.g. France for class type Country are called individuals.

Pick the corresponding cells in the source data which represent your individuals. Furthermore you can add specific metadata in order to describe your dataset in more detail. For example: Create a dataset for serveral countries, but just refering to women. If you want to refer to men as well you need to create another dataset.
